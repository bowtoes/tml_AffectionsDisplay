[h1]v0.1.0[/h1]
Adds text when hovering over an NPC head in the housing menu which lists their biome and neighbor preferences.
Comes with a configurable key to toggle the display on and off.

[img]https://steamuserimages-a.akamaihd.net/ugc/35567211346444023/404F6349CA7E77D9A9CE7C5D3632EB5B5B9160EB/[/img]

Implementation inspired by [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2687866031]Census[/url] and [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2857866907]NPC Happiness Viewer[/url]

The mod also makes it possible for other mods to query this information and the text descriptions, if it would be helpful.

Source code available: https://github.com/bowtoes/tml_PreferencesDisplay

(The icon is of Saint Christopher)
